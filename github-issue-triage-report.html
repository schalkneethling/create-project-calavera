Project Goal Summary

Calavera helps web developers compose, apply, inspect, and refresh repeatable project tooling recipes without coupling those recipes to a framework or starter. Current focus is the recipe-driven CLI, shared integration catalog, Vite-based composer, safe dry-run/JSON automation, package-manager compatibility, and stable generated tooling quality.
